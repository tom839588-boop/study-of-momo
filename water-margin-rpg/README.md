# 水浒英雄传 — 回合制 RPG

基于《水浒传》经典剧情改编的回合制角色扮演游戏。

## 特性

- 8 个剧情关卡，9 位梁山好汉
- 回合制战斗：攻击 / 技能 / 防御
- 每个英雄 2 个专属技能
- 水墨国风视觉，移动端优化
- 离线运行，无需网络

## 安装方式

### 方式一：浏览器直接玩（立刻可用）

打开浏览器访问页面即可游玩。

### 方式二：Android APK 安装

1. 前往 GitHub Actions 页面下载最新 APK 构建产物
2. 或者在本地构建：`cd android && ./gradlew assembleRelease`
3. 将 APK 传到华为手机安装

## 操作说明

- **攻击** — 选择目标进行普通攻击
- **技能** — 消耗 SP 释放英雄专属技能
- **防御** — 本回合减伤 60%
- 每关胜利后可获得新英雄加入队伍
